Hello! and welcome to the home page of this project.
This is a simple list manager of integers (we can only manage integers). Here are the different paths you can access and how to use them:

1.- /elements/
This path display us the elements that we actually have in our list

2.- /element/add/beginning/{int}
This path take an url parameter to insert an element at the beginning of the list (only integers or 404)
Example: /element/add/beginning/77

3.- /element/add/end/{int}
This path take an url parameter to insert an element at the end of the list (only integers or 404)
Example: /element/add/beginning/77

3.- /element/add/end/{int}
This path take a path parameter to insert an element at the end of the list
Example: /element/add/end/92

4.- elements/add/before/{int:old_num}/{int:new_num}/
This path take two url parameters:
  • old_num = The number in the list that we're using as a reference
  • new_num = The number that we want to insert before the referenced number

Example: /element/add/before/876/23

5.- elements/add/after/{int:old_num}/{int:new_num}/
This path take two url parameters:
  • old_num = The number in the list that we're using as a reference
  • new_num = The number that we want to insert after the referenced number


Example: /element/add/after/77/78
